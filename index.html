He/Him

Soy un diseñador gráfico de la ciudad de Hermosillo, Sonora. Mi enfoque principal es el diseño de páginas web, branding y publicidad. Siempre estoy buscando mejorar mis habilidades y aprender nuevas técnicas para llevar mis diseños al siguiente nivel.

QUEEN

"Se diseñó un logotipo, empaque y etiquetas para la miel natural, el producto principal de QUEEN, una marca de productos orgánicos de abeja. El resultado es una identidad visual coherente y atractiva para la marca y su producto."

ICABURE

"Se diseñó un isologo para ICABURE, un despacho de arquitectos, que refleja los módulos del hogar y muestra las iniciales de los arquitectos en orden. El resultado es una imagen visualmente atractiva y significativa para la marca."

PEQUE STYLE

"Para PequeStyle, una tienda de ropa de alta calidad para infantes, se creó un isologo único y se presentó una propuesta de interiorismo para la tienda, con el objetivo de ofrecer una experiencia de compra memorable a los clientes."

REVO

"Este isologo fue diseñado para REVO, una revista de moda, con una tipografía personalizada que combina elementos retro y modernos. El resultado es una identidad visual única y atractiva para la marca."
